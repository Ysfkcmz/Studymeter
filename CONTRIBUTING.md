# Contributing Guidelines

Vielen Dank für Ihr Interesse an der Weiterentwicklung von Studymeter! Wenn Sie Code beisteuern möchten, befolgen Sie bitte diese Schritte:

1. **Fork** das Repository.
2. Erstellen Sie einen neuen Branch für Ihr Feature (`git checkout -b feature/NeueFunktion`).
3. Committen Sie Ihre Änderungen (`git commit -m 'Feature XYZ hinzugefügt'`).
4. Pushen Sie den Branch auf GitHub (`git push origin feature/NeueFunktion`).
5. Erstellen Sie einen **Pull Request** zur Überprüfung.

Bitte stellen Sie sicher, dass Ihr Code den MISRA-C / C++ Standards entspricht und die FreeRTOS-Architektur nicht blockiert.
